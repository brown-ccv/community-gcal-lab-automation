1 Create Google Cloud Project

  1. Go to Google Cloud Console
  2. Click the project dropdown at the top
  3. Click "New Project"
  4. Name it: "Lab Calendar Automation" (or your choice)
  5. Click "Create"

2 Enable Google Calendar API

  1. In your project, go to "APIs & Services" ‚Üí "Library"
  2. Search for: Google Calendar API
  3. Click on it and click "Enable"

3 Configure OAuth Consent Screen

Before creating credentials, configure the OAuth consent screen:

  1. Go to "APIs & Services" ‚Üí "OAuth consent screen"
  2. Choose "External" (or "Internal" if using Google Workspace)
  3. Fill in required fields:
    • App name: Lab Calendar Automation
    • User support email: Your email
    • Developer contact: Your email
  4. Click "Save and Continue" through scopes
  5. On "Test users" page, click "+ Add Users"
  6. Add your email address
  7. Complete the wizard

4 Create OAuth Credentials

  1. Go to "APIs & Services" ‚Üí "Credentials"
  2. Click "+ Create Credentials" ‚Üí "OAuth client ID"
  3. Choose "Desktop app" as application type
  4. Name it: "Desktop Client"
  5. Click "Create"
  6. Click the download icon (‚¨áÔ∏è) next to your new client ID
  7. A JSON file will download

5 Add Credentials to Project

6 Restart the Server

After adding credentials.json:
